Mission Briefing

Phase 1: VS Code Extension Setup

Installing the GitHub Copilot Extension

First, let's get Copilot installed in VS Code:

VS Code Extensions Panel 
1. Open VS Code
2. Press Ctrl+Shift+X (Cmd+Shift+X on Mac) to open Extensions
3. Search for "GitHub Copilot"
4. Click "Install" on the official GitHub Copilot extension
5. Restart VS Code when prompted

Pro Tip

Also install the "GitHub Copilot Chat" extension for an enhanced conversational experience within VS Code!

Authentication Process

Now let's authenticate Copilot with your GitHub account:

  1. Trigger Authentication:

    Open any code file and start typing. Copilot will prompt you to sign in.

  2. Follow the Login Flow:

    Click "Sign in to GitHub" and follow the browser authentication process.

  3. Verify Access:

    Ensure your GitHub account has Copilot access (subscription required).

  4. Test Installation:

    Create a new JavaScript file and type // Function to calculate - you should see Copilot suggestions!

Phase 2: GitHub Integration

Web-Based Editor

Enable Copilot in GitHub's web interface:

  1. Navigate to Any Repository:

    Go to github.com and open any repository you have access to.

  2. Press the Period Key:

    Press . (period) to open the web-based VS Code editor.

  3. Verify Copilot Access:

    Look for the Copilot icon in the status bar. If not visible, your account may need Copilot access.

  4. Test Functionality:

    Edit a file and verify that Copilot suggestions appear as you type.

GitHub Codespaces

Copilot works seamlessly in Codespaces:

  1. Create a Codespace:

    From any repository, click "Code" → "Codespaces" → "Create codespace on main".

  2. Wait for Setup:

    The Codespace will automatically include Copilot if it's enabled on your account.

  3. Verify Integration:

    Open a code file and test that Copilot suggestions work normally.

Phase 3: Privacy and Settings

Configure Privacy Settings

It's important to understand and configure Copilot's privacy settings:

  1. Access Settings:

    In VS Code, go to File → Preferences → Settings (or Ctrl+,)

  2. Search for Copilot Settings:

    Type "copilot" in the search bar to find all Copilot-related settings.

  3. Review Key Settings:
    • github.copilot.enable - Enable/disable Copilot
    • github.copilot.inlineSuggest.enable - Inline suggestions
    • github.copilot.advanced - Advanced configuration options
  4. Configure for Your Needs:

    Adjust settings based on your organization's policies and personal preferences.

Important Privacy Note

Be aware of what code Copilot can see. For sensitive or proprietary code, consider:

  • Disabling Copilot for specific repositories
  • Using GitHub Copilot for Business for enhanced privacy controls
  • Reviewing your organization's AI tool policies
