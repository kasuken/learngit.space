Step 3: Creating Powerful Git Aliases

Let Copilot help you create time-saving Git aliases:

Alias Generation Chat 
// Ask Copilot to create aliases for common operations:

"Create Git aliases for these common operations:
1. Create a new feature branch from main
2. Quick commit with generated message
3. Push current branch to origin
4. Clean up merged branches
5. Interactive rebase of last 3 commits
6. Show pretty formatted log with graph"

Copilot-Generated Aliases:

Add to ~/.gitconfig 
[alias]
    # Branch management
    feat = "!f() { git checkout main && git pull && git checkout -b feature/$1; }; f"
    fix = "!f() { git checkout main && git pull && git checkout -b fix/$1; }; f"
    
    # Quick operations
    qc = "!git add . && git commit -m"
    pushup = "!git push -u origin $(git branch --show-current)"
    
    # Cleanup
    cleanup = "!git branch --merged main | grep -v main | xargs git branch -d"
    
    # History and logging
    lg = "log --graph --pretty=format:'%C(yellow)%h%Creset -%C(cyan)%d%Creset %s %C(green)(%cr) %C(blue)<%an>%Creset' --abbrev-commit"
    hist = "log --pretty=format:'%h %ad | %s%d [%an]' --graph --date=short"
